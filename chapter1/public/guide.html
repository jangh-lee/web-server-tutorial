Chapter 1 API Guide

학생들이 Postman 또는 curl로 바로 실습할 수 있도록 만든 간단한 API 안내 페이지입니다. 아래 예시 그대로 입력하면 `GET`, `POST`, `PUT`, `PATCH`, `DELETE` 흐름을 순서대로 연습할 수 있습니다.

메인 투두 페이지 현재 투두 JSON 보기 헬스체크 보기
Base URL http://localhost:3000
Content-Type application/json
실습 순서 추천 GET → POST → GET by ID → PUT → PATCH → DELETE
실시간 반영 열려 있는 페이지는 SSE로 자동 새로고침 없이 반영됩니다.

실습 순서

  1. `GET /api/todos`로 현재 목록을 확인합니다.
  2. `POST /api/todos`로 새 할 일을 추가합니다.
  3. 응답으로 받은 `id`를 사용해 `GET /api/todos/:id`를 호출합니다.
  4. `PUT /api/todos/:id`로 전체 필드를 수정합니다.
  5. `PATCH /api/todos/:id`로 일부 필드만 수정합니다.
  6. `DELETE /api/todos/:id`로 항목을 지웁니다.
GET /api/health

서버가 살아있는지 가장 먼저 확인합니다.

curl http://localhost:3000/api/health
GET /api/todos

현재 전체 할 일 목록을 조회합니다.

curl http://localhost:3000/api/todos
POST /api/todos

새 할 일을 만듭니다. Postman에서는 Body → raw → JSON으로 입력하면 됩니다.

{
  "title": "Postman으로 새 항목 만들기",
  "note": "학생용 POST 실습",
  "weather": "sunny",
  "plannedHour": "10",
  "plannedMinute": "30",
  "checkedHour": "",
  "checkedMinute": "",
  "done": false
}
curl -X POST http://localhost:3000/api/todos \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Postman으로 새 항목 만들기",
    "note": "학생용 POST 실습",
    "weather": "sunny",
    "plannedHour": "10",
    "plannedMinute": "30",
    "checkedHour": "",
    "checkedMinute": "",
    "done": false
  }'
GET /api/todos/:id

POST 응답에서 받은 `id`를 넣어 단건 조회합니다.

curl http://localhost:3000/api/todos/todo-12345
PUT /api/todos/:id

전체 필드를 한 번에 바꿉니다. `title`은 필수입니다.

{
  "title": "수정된 할 일 제목",
  "note": "PUT 전체 수정 예시",
  "weather": "cloudy",
  "plannedHour": "14",
  "plannedMinute": "00",
  "checkedHour": "14",
  "checkedMinute": "20",
  "done": true
}
curl -X PUT http://localhost:3000/api/todos/todo-12345 \
  -H "Content-Type: application/json" \
  -d '{
    "title": "수정된 할 일 제목",
    "note": "PUT 전체 수정 예시",
    "weather": "cloudy",
    "plannedHour": "14",
    "plannedMinute": "00",
    "checkedHour": "14",
    "checkedMinute": "20",
    "done": true
  }'
PATCH /api/todos/:id

일부 필드만 수정합니다. 완료 처리나 체크 시간 갱신 실습에 좋습니다.

{
  "done": true,
  "checkedHour": "16",
  "checkedMinute": "40"
}
curl -X PATCH http://localhost:3000/api/todos/todo-12345 \
  -H "Content-Type: application/json" \
  -d '{
    "done": true,
    "checkedHour": "16",
    "checkedMinute": "40"
  }'
DELETE /api/todos/:id

실습이 끝난 항목을 지웁니다. 성공 시 응답 코드는 `204 No Content`입니다.

curl -X DELETE http://localhost:3000/api/todos/todo-12345
